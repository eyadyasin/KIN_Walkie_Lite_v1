# KIN Walkie Lite v1.0

Private family voice communication app.

## What is KIN?

KIN Walkie Lite is a simple walkie-talkie style voice app for direct contacts and family groups.

## Features

- Direct voice messages
- Group voice messages
- Push notifications
- Background voice playback
- Online / offline presence
- Works without relying on real phone numbers
- Family-friendly lightweight design
